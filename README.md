# SendBird Solution Exercise - Chat 

## Introduction

Sendbird integration with ReactJs, which includes browser notification for files, text and admin.

## New install

- npx create-react-app geektopia
- cd geektopia
- npm install sendbird sendbird-uikit
- Customize!!!

## Current code

- yarn install
- yarn start
